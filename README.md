# ShiroPro Tools (Reborn)

城プロRE（御城プロジェクト:RE）の編成管理、バフ可視化、およびダメージ計算を行うためのWebアプリケーションです。
以前のツール開発で得た知見を活かし、**「視認性の高さ」**と**「パズルのような楽しさ」**を重視して再構築されました。

## 主な機能

### 1. 編成 & バフ・マトリックス
- **8枠の編成グリッド**: キャラの属性や武器種を直感的に表示。
- **バフ・マトリックス**: 誰がどのバフを持っているかを一覧表示。
    - **スタックバー**: 「自前特技」「味方バフ」「計略/鼓舞」の内訳を積み上げグラフで可視化。
    - **リファレンスゲージ**: 理想値に対して「不足」「適正」「過剰」を一目で判定。

### 2. アタッカー分析 & シミュレーション
- **詳細分析ビュー**: 特定のアタッカーにフォーカスし、ダメージ構成要素を分析。
- **What-If シミュレーション**: 「もし攻撃+30%があったら？」をワンクリックで試算し、伸び幅をプレビュー。
- **総合プロファイル**: DPSだけでなく、射程やコスト効率を含めたレーダーチャート表示。

### 3. Wikiデータインポート
- WikiのURLを入力するだけで、キャラクターデータを自動解析して取り込み。
- 複雑な特技テキストも正規表現エンジンで自動的に数値データ化。

## 技術スタック

- **Frontend**: React 19, TypeScript
- **Build Tool**: Vite
- **Styling**: TailwindCSS v4
- **Architecture**: Core (計算エンジン) と UI の完全分離

## 開発の始め方

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```
